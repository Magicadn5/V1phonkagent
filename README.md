# phonk_agent

Agent IA pour générer, analyser et préparer des morceaux IA.

## Objectif
- Générer une idée musicale.
- Créer un prompt pour Mureka.
- Lancer la génération.
- Télécharger le morceau.
- Vérifier la qualité audio.
- Générer une cover.
- Préparer les métadonnées.
- Préparer l’upload vers un distributeur.

## Installation
```bash
pip install -r requirements.txt
```
